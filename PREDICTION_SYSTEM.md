### 归一化公式（解决P0-2：量纲不统一）

每一层先独立归一化到0-100分，再按权重汇总。

```
第一步：各子模块独立评出原始分（各自量纲）
第二步：原始分 → min-max归一化 → 0-100分
        归一化分 = (原始分 - 理论最低分) / (理论最高分 - 理论最低分) × 100
第三步：层内加权
        硬数据层 = 基础实力×0.40 + 近10场状态×0.40 + 伤病折损×0.20
        外盘信号层 = 赔率结构×0.40 + 资金流向×0.33 + 盘口语言×0.27
        八卦风控层 = 更衣室×0.40 + 政治干扰×0.35 + 球星场外×0.25
第四步：跨层汇总
        最终总分 = 硬数据层×0.50 + 外盘信号层×0.30 + 八卦风控层×0.20
```

## 二、硬数据层（权重50%）—— 球队真实实力画像

### 2.1 球队基础实力（20分）

| 指标 | 满分 | 计算方式 |
|------|------|---------|
| FIFA排名 | 8分 | (49 - 排名) / 48 × 8，第1名得8.0分，第48名得0.17分 |
| Elo评分修正 | 4分 | Elo排名同上公式×4。FIFA与Elo各占权重，交叉验证亚非/欧美偏差 |
| 阵容总身价 | 5分 | 全队Transfermarkt身价，>10亿欧=5分，线性衰减至<5000万=0分 |
| 大赛淘汰赛经验 | 3分 | 近3届大赛(世界杯+洲际杯)进入淘汰赛：每次淘汰赛+1分，冠军额外+1分，上限3分 |

> **为什么引入Elo？** FIFA排名对亚洲/非洲球队有系统性低估（跨洲比赛权重分配偏差），Elo评分更新频率更高且跨洲权重更均衡。两套排名互为校验。

### 2.2 近10场战绩与状态评估（20分）★核心模块

#### 2.2.1 战绩面板（8分）

**数据范围定义：**
- 优先使用国家队近10场正式比赛（预选赛+洲际杯+热身赛）
- 热身赛权重×0.6（强度低于正赛）
- 俱乐部近5场数据仅用于2.2.4球员追踪，不计入国家队战绩面板

```
战绩原始分 = (近10场胜率×0.4 + 近5场胜率×0.6) × 10
```
近5场权重更高，体现"近期趋势优先"。

#### 2.2.2 对手质量加权 WARO指数（6分）★已修正

**修正后的公式（原版公式反了）：**

```
对手难度系数 = 1 + (48 - 对手FIFA排名) / 48

真实战斗力 = 战绩原始分 × 平均难度系数
```

**实例验证：**
| 对手 | FIFA排名 | 难度系数 | 解释 |
|------|---------|---------|------|
| 法国 | 2 | 1 + 46/48 = 1.96 | 赢强队获近2倍加成 ✓ |
| 阿根廷 | 1 | 1 + 47/48 = 1.98 | 最高难度 ✓ |
| 尼日利亚 | ~55(未晋级) | 1 + (-7)/48 = 0.85 | 赢弱队得分打折 ✓ |
| 佛得角 | ~45 | 1 + 3/48 = 1.06 | 弱队加成极低 ✓ |

**趋势修正交叉验证（解决P0-5）：**
```
趋势修正仅当对手质量可比时生效：
  if (近5场对手平均排名 ≤ 近6-10场对手平均排名 × 1.3):
      状态上行(近5胜率>近10胜率) → ×1.15
      状态下行(近5胜率<近10胜率) → ×0.85
  else:
      忽略趋势修正（胜率变化可能来自对手变弱，而非自己变强）
```

#### 2.2.3 攻防效率面板（6分）

```
进攻锐度 = 近5场场均进球 / 近10场场均进球
  >1.2=火力上升 +2分 | 0.8-1.2=平稳 +1分 | <0.8=进攻哑火 0分

防守稳固度 = 近5场零封率 / 近10场零封率
  >1.2=防线改善 +2分 | 0.8-1.2=平稳 +1分 | <0.8=防守退化 0分

场均净胜球 = 近5场(进球-失球)/5
  >1.5 → 2分 | 0.5-1.5 → 1分 | <0.5 → 0分
```

**倒扣预警（从上述总分中扣除）：**
- 近3场运动战零进球 → -1分（锋线结构性危机）
- 近3场场均被射正≥6次 → -1分（防线形同虚设）
- 近5场领先被逆转≥2次 → -2分（心理韧性存疑）
- 近3场全部靠点球/定位球得分 → -1分（运动战创造力枯竭）

#### 2.2.4 关键球员状态追踪（弹性±2分）

追踪每队3个核心位置球员的个人近期效率（国家队+俱乐部合并）：

| 追踪位置 | 评估指标 | 加减分 |
|---------|---------|--------|
| 头号射手 | 近5场进球效率 vs 赛季均值 | >120% +1分，<80% -1分 |
| 核心中场 | 近5场关键传球/创造机会 vs 赛季均值 | >120% +1分，<80% -1分 |
| 主力门将 | 近5场扑救成功率 vs 赛季均值 | >110% +1分，<90% -1分 |

> 弹性修正范围：-2到+2分。宁可空置不用，也不强行打分。

## 三、外盘信号层（权重30%）—— 钱是最诚实的预言 ★新增层

> **核心理念：** 博彩市场是全世界最聪明的群体智慧集合。每一分钱的流动背后都有信息。我们不看赔率"谁热"，而看赔率"怎么变"、资金"往哪走"、盘口"说什么"。

### 3.1 赔率结构与变化（12分）

#### 3.1.1 夺冠赔率排名（4分）

```
赔率排名分 = (球队总数 - 赔率排名 + 1) / 球队总数 × 4
```

#### 3.1.2 赔率漂移分析（5分）★核心指标

赔率的变化方向和速度比赔率绝对值更有信息量：

| 漂移类型 | 定义 | 信号含义 | 加减分 |
|---------|------|---------|--------|
| 急剧收缩 | 1周内赔率缩短≥20% | 内部信息流入（训练表现好/伤病恢复/签运好） | +2 |
| 温和收缩 | 1周内缩短5-20% | 市场逐渐看好 | +1 |
| 稳定 | 变化<5% | 市场共识稳定 | 0 |
| 温和外扩 | 1周内拉长5-20% | 市场信心减弱 | -1 |
| 急剧外扩 | 1周内拉长≥20% | 负面信息泄露（训练冲突/伤病隐瞒） | -2 |

#### 3.1.3 多庄家赔率一致性（3分）

```
赔率离散度 = 6大庄家(365/BetMGM/DraftKings/FanDuel/Caesars/Betfair)赔率标准差/均值

离散度<5%  → 3分（市场高度共识，置信度高）
离散度5-10% → 2分（正常分歧）
离散度10-20% → 1分（明显分歧，有庄家在接筹码）
离散度>20% → 0分（市场混乱，信息不对称严重）
```

### 3.2 资金流向分析（10分）

> **核心逻辑：** 赔率变化可能是庄家主动调整（平衡账面），也可能是大量资金涌入导致。只有区分这两种情况，才知道赔率变化有没有信息含量。

#### 3.2.1 投注量占比（4分）

```
某队投注量占全部48队的比例 → 反映公众热度

投注量占比>10% → 过热信号（公众情绪可能非理性）
投注量占比3-8% → 健康区间
投注量占比<1% → 被市场忽略（可能有价值洼地）
```

#### 3.2.2 大额资金（Sharp Money）追踪（4分）

**关键区分：公众的钱 vs 聪明钱**

```
Sharp Money信号（通过赔率变化+交易量异常检测）：
- 赔率在低交易量时剧烈变化 → 有内部信息
- 赔率在高交易量时稳定 → 市场有效定价
- 弱队方向出现异常大额投注 → 可能有冷门
```

#### 3.2.3 责任指数（Liability Index）（2分）

```
庄家责任 = 某队夺冠的总赔付风险 / 庄家该市场总风险池

责任指数高 → 庄家最怕这支球队赢 → 这支球队确实危险
```

### 3.3 亚洲盘口语言（8分）

> 亚洲盘口的精妙之处在于：它不为预测输赢，而是让你选择"相信哪边的叙事"。

#### 3.3.1 小组赛单场盘口分析

```
盘口深度解读：
  强队让2.5球以上 → 市场预期屠杀级差距
  强队让1-1.5球 → 正常差距，一球小胜可能
  强队让0.5球以下 → 市场认为有冷门风险
  强弱不分/平手盘 → 市场认为双方实力接近
```

#### 3.3.2 盘口水位变化

```
水位变化（赔率不变但赔付比例调整）：
  强队水位持续走低 → 资金不断涌入强队方向 → 市场非常确定
  强队水位走高 → 庄家在诱导/平衡 → 冷门可能
  水位剧烈波动 → 有新信息进入市场
```

#### 3.3.3 大小球盘口

```
大小球盘口 = 市场预期总进球数

盘口≥3球 → 预期高比分对攻战
盘口2-2.5球 → 正常
盘口≤1.5球 → 预期沉闷防守战 → 弱队可能摆大巴
```

## 五、输赢/比分输出引擎 ★最终产出

> 三层数据 → 总分差 → 胜平负概率 → 预期进球 → 具体比分。每一步都可追溯。

---

### 5.1 数据汇聚：两队最终总分

```
球队A最终总分 = 硬数据_A×0.50 + 外盘_A×0.30 + 八卦_A×0.20
球队B最终总分 = 硬数据_B×0.50 + 外盘_B×0.30 + 八卦_B×0.20

总分差 Δ = 球队A总分 - 球队B总分
```

八卦层注意：八卦是扣分制（满分100，向下扣减），所以八卦得分越低=风险越大。

---

### 5.2 总分差 → 胜平负概率（泊松模型）

#### 5.2.1 基础预期进球计算

```
球队A预期进球 xG_A = 联赛基准均值 × (1 + 总分优势系数)

基准均值 = 1.35（世界杯历史场均进球的中位数）

总分优势系数 = (球队A总分 - 50) / 50
  → 总分=80的强队：系数=0.60，xG=1.35×1.60=2.16
  → 总分=50的平庸队：系数=0.00，xG=1.35×1.00=1.35
  → 总分=30的弱队：系数=-0.40，xG=1.35×0.60=0.81
```

#### 5.2.2 对手压制修正

```
球队A的最终xG = xG_A × 对手防守压制系数

对手防守压制系数 = 1 - (球队B防守分 - 50) / 200
  → 对手防守极强(B防守分80)：系数=0.85，xG打85折
  → 对手防守正常(B防守分50)：系数=1.00，xG不变
  → 对手防守极弱(B防守分20)：系数=1.15，xG加成15%
```

**防守分来源：** 2.2.3节攻防效率面板中的"防守稳固度"指标 + 2.3伤病折损中防线核心缺阵扣分。

#### 5.2.3 泊松分布 → 胜平负概率

根据两队最终预期进球xG_A和xG_B，用泊松模型计算所有可能比分的概率：

```
P(A进k球) = (xG_A^k × e^(-xG_A)) / k!
P(B进k球) = (xG_B^k × e^(-xG_B)) / k!

截断到0-8球（超过8球的概率<0.1%，忽略）

全概率矩阵求和：
  P(A胜) = Σ(i>j) P(A进i球)×P(B进j球)    for i,j in 0..8
  P(平局) = Σ(i=j) P(A进i球)×P(B进j球)
  P(B胜) = Σ(i<j) P(A进i球)×P(B进j球)
```

**示例（法国 vs 塞内加尔）：**
```
法国xG=2.10, 塞内加尔xG=0.85
→ P(法国胜)=67.1%  P(平)=20.3%  P(塞内加尔胜)=12.6%
```

#### 5.2.4 八卦冷门修正

八卦信号作为独立的概率偏移量，直接作用于泊松结果：

```
修正后P(冷门) = P(冷门) + 八卦偏移量

八卦偏移量计算：
  热门方八卦扣分 - 冷门方八卦扣分  →  转化为概率偏移

  差值≥10分 → P(冷门)+8%，P(平)+2%，P(热门)-10%
  差值5-9分 → P(冷门)+5%，P(平)+1%，P(热门)-6%
  差值1-4分 → P(冷门)+2%，P(平)+1%，P(热门)-3%
  差值<1分  → 不调整
```

**边界保护：** 修正后胜平负概率之和=100%。如果冷门方胜率被推至>30%但硬数据不支持，系统标注"⚠️冷门信号与数据矛盾，以数据为准"。

---

### 5.3 最可能比分 + 比分概率分布

#### 5.3.1 最可能比分选取

从泊松全概率矩阵中，选取概率最大的3个比分：

```
概率矩阵中直接取Top-3 (i,j)组合:
  P(i,j) = P(A进i球)×P(B进j球)

Top-1 = 最可能比分
Top-2/3 = 备用比分
```

#### 5.3.2 比分合理性过滤

以下比分组合在职业足球中概率极低，手动压低概率权重×0.5：
- 双方均≥4球（如4-4, 5-4等）
- 一方0球另一方≥6球
- 一方≥7球

#### 5.3.3 大小球盘口约束

如果外盘大小球盘口与模型预测的总进球数偏差>1球，触发约束：

```
模型总进球 = xG_A + xG_B
盘口总进球 = 大小球盘口值

如果Abs(模型-盘口)>1球：
  → 以盘口为准，将双方xG等比例缩放至总进球=盘口值
  → 原因：市场规模数十亿美元，进球预期通常比模型更准
```

### 5.5 特殊场景处理

#### 场景1：淘汰赛（含加时和点球）

淘汰赛必须有胜负，平局概率需要拆分为加时胜和点球胜：

```
淘汰赛90分钟内：
  胜平负概率同小组赛算法

如果平局（进入加时）：
  加时赛简化处理：双方xG各×0.3（加时体能下降，进球率约为常规时间30%）
  加时仍平 → 进入点球大战

点球大战胜率：
  基于门将扑点率 + 罚球手历史命中率 + 球队点球大战历史胜率
  → 独立子模块输出
```

#### 场景2：实力悬殊比赛（总分差>25）

```
触发条件：Δ>25分（如法国 vs 约旦级别差距）

处理：
- 泊松模型在极端xG差距下可能失真
- 引入"屠杀上限"：xG差值>3.5时，弱队xG强制不低于0.3（尊严进球概率）
- 比分分布更分散：大比分（3-0, 4-0, 4-1）概率上升
```

#### 场景3：小组赛末轮（出线形势影响）

```
已知条件：双方出线形势（必须赢/平即可/已出线/已淘汰）

动力修正系数：
  必须赢的球队 → xG×1.15（进攻欲望↑）
  已出线轮换阵容 → xG×0.80（战意↓）
  已淘汰球队 → 看球队性格（有的崩溃×0.70，有的放松反而发挥好×1.0）
  双方都需要平局 → 平局概率×1.5（默契球风险）
```

## 十三、回测验证框架

### 准确率定义

| 预测类型 | 评估指标 | 基准线 |
|---------|---------|--------|
| 小组出线预测 | 24/32正确率 | 随机=50%，赔率热门=~75% |
| 单场胜平负 | 48场小组赛正确率 | 随机=33% |
| 淘汰赛晋级 | 16场比赛逐轮正确率 | 随机=50% |
| 八卦预判 | 5条各自可证伪判定结果 | - |
| 比分预测 | 准确比分+净胜球方向 | 需积累样本后评估 |

### 赛季末审计

每轮淘汰赛后更新预判追踪表，决赛后出具完整准确率报告。目标：**关键拐点正确率高于赔率市场隐含概率5个百分点以上。**

---

> **设计哲学 v2.0：** 足球预测永远不可能100%准确。我们的目标是三重优势——硬数据告诉我们"谁更强"（精度优势），外盘信号告诉我们"钱怎么走"（信息优势），八卦风控告诉我们"谁可能崩"（风险优势）。三者叠加，在关键决策点的正确率持续高于市场平均水平，就是这套系统的全部意义。
