## 二、优化1：平局概率修正 🔴最高优先级

### 2.1 根因分析

模型使用泊松分布，平局概率由 xG_A 与 xG_B 的接近程度决定。当前问题：

- 模型给平局的概率在19%-28%之间，均值≈23%
- 实际首轮平局率50%
- **泊松模型本身不会系统低估平局，问题出在xG输入层**

### 2.2 发现的可参考数据

| 数据点 | 值 | 来源 |
|--------|-----|------|
| 本届首轮平局率 | 8/16 = 50% | 实际赛果 |
| 世界杯历史首轮平局率 | ~28% | 1998-2022统计 |
| 48队扩军后首轮平局率 | 无历史数据 | 首次48队赛制 |
| 补水暂停制度 | 本届新增 | ESPN/Van Dijk批评报道 |
| 高温城市比赛 | Dallas 34°C, Houston 35°C, Miami 33°C | 气象数据 |
| 弱队摆大巴倾向 | Cape Verde 0-0 Spain, Iran 2-2 NZL | 弱队首轮生存策略 |

### 2.3 优化方案

**A. 首轮平局加成系数（立刻生效）**

```
小组赛首轮平局基准概率 × 1.6
小组赛次轮平局基准概率 × 1.3
小组赛末轮平局基准概率 × 1.0（恢复正常）
淘汰赛平局基准概率 × 0.8（必须分胜负）
```

依据：首轮球队保守试探，48队赛制下弱队"拿1分就是胜利"心态更普遍。

**B. 温差-平局关联表**

| 比赛温度 | 平局修正系数 | 依据 |
|---------|------------|------|
| >32°C | ×1.3 | 高温降低比赛节奏，补水暂停打断进攻连续性 |
| 25-32°C | ×1.1 | 轻微影响 |
| <25°C | ×1.0 | 正常节奏 |

**C. 实力悬殊-平局悖论**

当强队总分 > 85 且 弱队总分 < 50 且 弱队防守分 > 55 时：
→ 平局概率额外 +5%
→ 原因：强队围攻无果 + 弱队全员退守 = 0-0或1-0概率上升

（España 0-0 Cape Verde 就是这个模式的典型案例）

### 2.4 预期效果

应用A+B+C后，平局预测概率从23%均值提升至约33%，与实际50%仍有差距但大幅改善。

## 三、优化2：大比分屠杀死上限 🟡中优先级

### 3.1 根因分析

`apply_handicap_constraint` 中的 `market_goals` 公式过于保守：

```python
market_goals = (r5_h["gf"] + r5_h["ga"] + r5_a["gf"] + r5_a["ga"]) / 10
```

当双方近5场场均总进球=3.2球时，这个公式输出3.2，但强弱悬殊时强队进球被弱队低失球拉低均值。

### 3.2 发现的可参考数据

| 比赛 | 模型预测 | 实际 | 弱队近5场失球 | 强队近5场进球 |
|------|---------|------|-------------|-------------|
| GER-CUW | 2-1 | 7-1 | CUW 14球 | GER 18球 |
| SWE-TUN | 2-1 | 5-1 | TUN 12球 | SWE 15球 |
| USA-PAR | 2-1 | 4-1 | PAR 9球 | USA 13球 |

**规律：当强队场均进球 > 2.5 且 弱队场均失球 > 2.0 时，屠杀概率显著上升。**

### 3.3 优化方案

**A. 屠杀因子**

```python
slaughter_factor = 1.0

strong_team_goals_per_game = r5_strong["gf"] / 5
weak_team_conceded_per_game = r5_weak["ga"] / 5

if strong_team_goals_per_game > 2.5 and weak_team_conceded_per_game > 2.0:
    slaughter_factor = 1.4  # 强队xG上浮40%
elif strong_team_goals_per_game > 2.0 and weak_team_conceded_per_game > 1.5:
    slaughter_factor = 1.2
```

**B. 取消大比分过滤的过度压降**

当前 `build_prob_matrix` 中：
```python
if (a >= 4 and b >= 4) or (a == 0 and b >= 6) or (b == 0 and a >= 6):
    p *= 0.5  # 当前：一律对折
```

改为：
```python
if (a >= 4 and b >= 4) or (a == 0 and b >= 6) or (b == 0 and a >= 6):
    p *= 0.7  # 折30%而非50%
```
并新增：如果差距>30分，不应用此过滤。

### 3.4 预期效果

GER-CUW 预测从 2-1 修正为 3-1 或 4-1；SWE-TUN 从 2-1 修正为 3-1。虽仍低于实际7-1/5-1，但方向改善。

## 四、优化3：微小分差方向判定 🟡中优先级

### 4.1 根因分析

当 Δ < 8 时，模型给出的方向判断接近于抛硬币。

| 比赛 | Δ | 模型方向 | 实际 | 胜/平/负分布 |
|------|---|---------|------|------------|
| NED-JPN | +4.1 | 日本胜 | 平局 | 35/24/41 |
| CIV-ECU | -3.7 | 厄瓜多尔 | 科特迪瓦 | 33/31/36 |
| BEL-EGY | +11.4 | 平手盘 | 平局 | 37/26/37 |
| IRN-NZL | +5.8 | 伊朗 | 平局 | 39/25/36 |

### 4.2 优化方案

**输出层增加"置信度标签"：**

```
Δ > 15  → "高置信度"
Δ 8-15  → "中置信度"  
Δ < 8   → "低置信度 — 建议观望"
```

当 Δ < 8 时，系统输出应强制标注"本场方向预测置信度低，胜平负接近随机，建议关注比分和大小球而非胜负"。

**评分层面不调整**——这些比赛本质就是难预测的，强行修正方向只会引入新偏差。正确的做法是诚实标注不确定性。

## 五、优化4：球队实力评估修正 🟢低优先级

### 5.1 发现的具体偏差

| 球队 | 模型问题 | 实际表现 | 修正方向 |
|------|---------|---------|---------|
| **Australia** | 排名第12（79.0分），严重低估 | 2-0土耳其，Irankunda爆发，Beach 8扑救 | +5分硬数据，提升近5场权重 |
| **Scotland** | 排名第24（69.5分），略低估 | 1-0海地，McGinn致胜 | +3分，1998年后首胜士气加成 |
| **Tunisia** | 排名第42（53.8分），高估 | 1-5惨败瑞典，主帅将被解雇 | -5分，八卦层需加Lamouchi下课信号 |
| **Saudi Arabia** | 排名第43（53.6分），高估 | 1-1乌拉圭表现出色 | +3分，赛前换帅反向激励未被捕捉 |

### 5.2 优化方案

**Australia案例揭示的问题：** `recent_form` 中近5场权重已是0.6，但Australia近5场4胜0平1负的表现仍被近10场稀释。建议：

```python
# 当前：近10场权重0.4，近5场权重0.6
# 优化：引入"爆发性状态"检测
if 近5场胜率 > 近10场胜率 + 0.2:
    近5场权重 = 0.75  # 进一步倾斜
```

**Tunisia案例：** 赛前内乱的八卦信号（Lamouchi将被解雇）未能及时录入gossip.json。建议：

```
八卦数据刷新频率从"持续监控"提升到"赛前24小时强制刷新"
每场比赛前自动拉取双方最新新闻标题做关键词匹配
```

## 八、回测验证目标

第二轮（12场比赛）应用P0+P1优化后，目标：

- 方向正确率：50% → **65%+**
- 平局预测覆盖率（实际平局中被模型Top1预测为平局的比例）：当前0/8 → **3+/预计平局数**
- 大比分（净胜3+）Top3覆盖：当前1/3 → **2/3+**
- 冷门预警命中率：维持4/9或更优
